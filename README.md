# FlowTrack

A responsive landing page for a fictional SaaS project management tool, built with React, Vite, and Tailwind CSS.

## Sections

- Hero with gradient text and CTAs
- Features grid (AI, Collaboration, Workflows, Dashboards, Security, Mobile)
- Stats bar (10k+ teams, 99.9% uptime, etc.)
- Pricing table (Starter, Professional, Enterprise)
- Testimonials
- Call to action
- Footer with links and social icons

## Stack

- **React 18** — UI components
- **Vite 8** — build tool
- **Tailwind CSS 3** — styling
- **Lucide React** — icons
- **Vercel** — deployment

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```
